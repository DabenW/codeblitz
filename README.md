## 项目启动
### 准备
需要安装 yarn >= 1.0，使用 yarn 的 [workspaces](https://classic.yarnpkg.com/en/docs/workspaces/) 来管理 packages

### 启动项目
```bash
1. yarn
2. yarn run init
3. yarn run dev
```
终端打开输出的地址，默认端口 9009，如被占用，会从 9009 查找可用端口
